:::tip[Pesde Support]

Pesde can also install Reheaven directly from the Wally registry.

```sh
pesde add wally#wiindsom/reheaven
```

To install a specific version:

```sh
pesde add wally#wiindsom/reheaven@1.0.0
```

:::
